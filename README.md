## Personal Portfolio Website


#### Project Description :

This project is a Personal Portfolio Website created using HTML and CSS to showcase personal details, skills, projects, and contact information.
It is designed for beginners to present their profile professionally and serves as an online resume and personal branding platform.
#### Purpose of the Project :

Create an online presence for a web developer

Showcase skills, projects, and resume in one place

Practice HTML structure and external CSS styling

Build a beginner-friendly portfolio for internships or jobs
#### Features :

Clean and simple layout

About Me section with profile image

Skills listing

Projects showcase

Resume download option (PDF)

Contact details section

Footer with copyright information
#### Technologies Used :

HTML5 – Structure and content

CSS3 – Styling and layout (external stylesheet)
